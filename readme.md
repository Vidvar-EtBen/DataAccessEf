# EtBen.DataAccess.EF.Sql

A lightweight, extensible .NET 9 library implementing the **Repository** and **Unit of Work** patterns using **Entity Framework Core**.

---

## Overview

`EtBen.DataAccess.EF.Sql` provides a structured, maintainable approach to data access by offering:

- Generic repository interfaces and base classes  
- A repository factory for dependency injection  
- A unit of work for transactional consistency  

This library promotes **clean separation of concerns**, making your codebase **testable** and **extensible**.

---

## Features

- **CRUD operations for any entity type**  
- **Generic repository interfaces** for consistent data access patterns  
- **Repository factory** to simplify dependency injection  
- **Unit of Work** implementation for transactional integrity  
- Designed for **clean architecture** and **testable code**
